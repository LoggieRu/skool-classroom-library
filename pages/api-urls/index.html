üî® Breaking It Down

1

BASE URL (The Foundation)

https://api.openweathermap.org/data/2.5

This is like your house address ‚Äî it never changes

2

ENDPOINT (What you want)

This tells the API what data you want (weather, forecast, etc.)

3

PARAMETERS (Specific details)

?q={city}&appid={your_key}

The stuff in {} brackets gets REPLACED with real values
